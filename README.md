## ClassConnect Backoffice (Admins Only)

### What You'll Find

- `USERS` table
- `COURSES` table
- `AUXILIARY TEACHERS` table
- General stats
- Platform Logs
- Platform Metrics

### How To Run

To run locally you'll need React 18.2.0

```bash
npm run start
```

Or you can just visit the hosted site at: https://backoffice-rbyl.onrender.com/dashboard
